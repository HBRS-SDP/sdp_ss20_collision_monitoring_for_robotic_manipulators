10 ## Description
11 This repository is a group submission for MAS Software Development Project. The
12 goal of the project is to implement a given software algorithm from a research
13 paper, while communicating with a client to add desired features. The paper
14 implemented is a method of obstacle avoidance for robotic manipulators and
15 at the request of the client a combination of 2 papers was implemented and can
16 be found in the reference section \[1\]\[2\]
17 
18 ## Structure
19 This repository is structured to contain multiple implementations/libraries and
20 documents for the final submission. The repository has a hierarchial structure
21 based around the Collision Monitoring library leading to the final ROS package
22 used for demonstration. The following links list the core functionalities of
23 this repository.
24 
25 ### Contents
26 - [Collision Monitoring Library](http://github.com/broccan/sdp_ss20_collision_monitoring_for_robotic_manipulators/blob/master/collision_monitoring/README.md):
27  The base library designed to be highly portable and used for any manipulator type
28 - [Kinova Arm Package](http://github.com/broccan/sdp_ss20_collision_monitoring_for_robotic_manipulators/blob/master/src/README.md):
29  The test implementation of the Collision monitoring library designed for a
30  Kinova gen3 7DOF manipulator.
31 - [Collision Avoidance ROS Package](http://github.com/broccan/sdp_ss20_collision_monitoring_for_robotic_manipulators/blob/master/catkin_workspace/README.md):
32  The implementation of the obstacle avoidance and tracing algorithm that the
33  base library is designed to suit.
34 - Documentation & other related works:
35  The last section found in the /deliverables directory contains the paper,
36  presentation and API documentation which is submitted along with this code.
