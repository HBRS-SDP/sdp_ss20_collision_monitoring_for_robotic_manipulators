Detailed Description

Arm controller class to demonstate the use of the Collision monitoring library.

The class uses the monitor class from the custom library to control a 7DOF kinova arm implementing the paper: [1]H. Hoffmann, P. Pastor, D.-H. Park, and S. Schaal, “Biologically-inspired dynamical systems for movement generation: Automatic real-time goal adaptation and obstacle avoidance,” in 2009 IEEE International Conference on Robotics and Automation, Kobe, May 2009, pp. 2587–2592, doi: 10.1109/ROBOT.2009.5152423.
